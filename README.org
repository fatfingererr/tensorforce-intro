*** TensorForce 簡介

- 開源的 Reinforcement Learning (RL) library

- 建立在 TensorFlow 上，目前與 Python 2.7 和 3.5 相容

- 沒有對輸入和狀態有限制，可自由建構代理與環境

- 在 Agent 和 Update logic 做了嚴格分離，以便在真實環境中使用

- 主打將 RL logic 呈現在 TensorFlow 的圖表上頭，減少對 Python 的依賴
